## ✨ Features

- 📄 Upload any PDF (college notes, textbooks, slides)
- 🔍 Keyword-based retrieval across all document chunks
- 🤖 Answers generated by Gemini 2.0 Flash
- 💬 Chat interface with message history
- 🔒 PDF never uploaded to any server — 100% browser-based
- ⚡ Zero installation — single HTML file

---

## 🏗️ Architecture

```
PDF Upload
    │
    ▼
PDF.js (extract text)
    │
    ▼
Chunker (400 words, 60 word overlap)
    │
    ▼
Keyword Scorer (TF scoring per chunk)
    │
    ▼
Top-5 Chunks Retrieved
    │
    ▼
Gemini 2.0 Flash API
    │
    ▼
Answer displayed in chat
```

## ⚡ Quick Start

### Option 1 — Web App (Recommended, Zero Install)

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/notesmind.git

# 2. Open in browser
open src/index.html
# OR just double-click src/index.html
```

Then:
1. Get a free Gemini API key from [aistudio.google.com](https://aistudio.google.com)
2. Paste the key in the sidebar
3. Upload your PDF notes
4. Click **"Index My Notes"**
5. Ask questions!

---

### Option 2 — Python / Streamlit Version

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/notesmind.git
cd notesmind

# 2. Install dependencies
pip install streamlit google-generativeai pypdf numpy

# 3. Run
streamlit run src/streamlit_app.py
```

## 🔑 Getting a Gemini API Key

1. Go to [aistudio.google.com](https://aistudio.google.com)
2. Sign in with your Google account
3. Click **"Get API Key"** → **"Create API Key"**
4. Copy the key (starts with `AIza...`)
5. Paste it into the NotesMind sidebar

> Free tier: 15 requests/minute, 1500 requests/day — enough for study use.

## 🧠 How It Works

### 1. PDF Text Extraction
PDF.js reads the uploaded file entirely in the browser. No server upload.

### 2. Chunking
Text is split into 400-word chunks with 60-word overlap to preserve context at boundaries.

### 3. Retrieval
When you ask a question, each chunk is scored by counting how many question words appear in it (Term Frequency scoring). Top 5 chunks are selected.

### 4. Generation
The top 5 chunks + your question are sent to Gemini 2.0 Flash with a strict prompt:
> "Answer ONLY from the notes below. If not found, say so."

This prevents hallucination.

## 📊 Limitations & Future Improvements

| Current | Improvement |
|---------|-------------|
| Keyword search | Semantic embeddings (sentence-transformers) |
| Browser-only | Python backend (FastAPI) |
| Single PDF | Multi-document support |
| No citations | Show source page number |
| No persistence | ChromaDB / FAISS vector store |
| Free tier limits | Paid API or local LLM (Ollama) |

## 🆚 vs Full Production RAG

| Component | NotesMind | Production RAG |
|-----------|-----------|---------------|
| Framework | None | LangChain / LlamaIndex |
| Embeddings | Keyword TF | text-embedding-3-small |
| Vector DB | Browser array | ChromaDB / Pinecone |
| Retrieval | TF scoring | Cosine similarity + Reranking |
| Backend | None | FastAPI |
| Hosting | Local HTML | AWS / Vercel |
