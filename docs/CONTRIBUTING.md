# 🤝 Contributing to NotesMind

## Ways to Contribute

- Fix bugs
- Add semantic search with embeddings
- Add multi-PDF support
- Add citation / source page numbers
- Improve the UI
- Add support for other file types (DOCX, TXT)

## Setup

```bash
git clone https://github.com/yourusername/notesmind.git
cd notesmind
# Open src/index.html in browser to test
```

## Pull Request Process

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make changes
4. Test in Chrome
5. Submit PR with description of what you changed and why
